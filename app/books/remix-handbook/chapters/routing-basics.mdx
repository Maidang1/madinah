# 文件命名与路由映射

Remix 的路由由 `app/routes` 下的文件结构直接决定。利用下划线与点语法可以创建布局、索引与动态段。

```
app/routes/
├── _landing._index/route.tsx   → /
├── app._index/route.tsx        → /app
└── app_.projects.$id/route.tsx → /app/projects/:id
```

通过共享布局组件，我们可以在父路由中加载一次昂贵的数据，并在子路由中消费。
