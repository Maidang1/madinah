# oh-my-opencode

## What is oh-my-opencode?

`oh-my-opencode` is a plugin for `opencode`. It collects the shortcuts, defaults, and small workflow improvements I use most often, then packages them in one place so a fresh `opencode` setup feels ready to use faster. This note is still a work in progress. Upcoming sections will cover the plugin surface, configuration shape, and pieces worth borrowing for other coding-agent tools.
