## What is oh-my-opencode?

`oh-my-opencode` is a plugin for `opencode`. It collects the shortcuts, defaults, and small workflow improvements I use often, then packages them into one place so a new `opencode` setup feels ready to use sooner.

This note is still a work in progress. The next sections will cover the plugin surface, configuration shape, and the parts worth borrowing for other coding-agent tools.
